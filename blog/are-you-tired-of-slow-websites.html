Are you tired of slow websites

Joshua Matambo
|
October 15, 2020

Have you ever visited a slow website that takes what seems to be forever to load. In a world that we receive all our content in real time no one has the time and the patience to wait to receive their content. They are may reasons that a website might be slow to load a few common ones are poor web design when assets of a large size are used within the project and using one central server where the speed of accessing the website is determined by the distance from the server.

To solve these two common problems better web design can always help using assets of smaller size and for the second problem using a content distribution network which is offered by services like Amazon web services this system uses multiple servers around the world to make accessing content faster as the information is always close to the user. Likely Webflow comes with a built in functionality that helps the web developer affect both of these problems by limiting the size of pictures to 4mb which is still to large in my opinion and uses a central distribution network
